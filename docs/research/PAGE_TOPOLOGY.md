# 4D Smart Menu Page Topology

1. Top nav with restaurant logo and four circular action buttons.
2. Hero block with looping background video, greeting label, restaurant title, and status pill.
3. Section label row for categories plus animated swipe hint.
4. Horizontal category scroller with icon cards.
5. Sticky item controls header containing search, category title, diet filters, and layout switcher.
6. Main item list/grid area with dish cards and route-out to detail view.
7. Floating waiter-call action button.
8. Full-screen overlay for modal and cart states.
9. Center waiter popup.
10. Bottom cart sheet.
11. Center toast message for waiter/order success.
